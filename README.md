# 嬰兒哭聲偵測播放器

開啟麥克風持續監聽，偵測到寶寶哭聲或哀叫時，自動播放指定的安撫音檔。
純前端、零依賴，麥克風聲音只在本機即時分析，不錄音、不儲存、不上傳。

- **線上使用**：https://erich0793.github.io/claude-on-the-net/
- **使用說明**（操作步驟、設定建議、疑難排解）：`guide.html`（頁面右上角「📖 使用說明」）

## 主要功能

- **雙模式偵測**：僅哭泣（持續哭聲累積）／哭泣＋哀叫（20 秒內 2 聲合格短聲）
- **聲學判定**：自相關法量測基頻（嬰兒 250–1000Hz），自動排除大人講話（85–255Hz）；
  音量門檻相對背景噪音自動校正，開始監聽含 3 秒環境校準
- **音檔管理**：本機記憶（IndexedDB）、瀏覽器內剪輯（位元組切片解碼，大檔不爆記憶體）、
  下載匯出（跨裝置共用）
- **播放控制**：循環播放至指定時長、哄睡音量淡出（Web Audio 增益）、手動停止後 30 秒冷靜期
- **iOS 韌性**：自動播放解鎖、螢幕喚醒鎖、麥克風中斷看門狗與自動重連、
  播放前先釋放麥克風（確保路由到藍牙／AirPlay 音響且不劣化音質）
- **音響支援**：AirPlay 相容模式（繞過 Web Audio）、脈衝式保持連線（每 4 分鐘發送 3 秒
  17.5kHz 訊號防閒置斷線；錄音期間 iOS 不會將網頁音訊送往外部音響，故採脈衝設計）
- **事件紀錄**：觸發、短聲、麥克風狀態、診斷統計皆留時間戳（localStorage），供隔日回顧與調校

## 本機開發

```bash
python3 -m http.server 8000
# 開啟 http://localhost:8000（麥克風僅在 https/localhost 可用）
```
