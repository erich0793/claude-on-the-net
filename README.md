# 嬰兒哭聲偵測播放器

開啟麥克風持續監聽，偵測到疑似嬰兒哭聲時自動播放你指定的音檔（例如安撫音樂）。

## 快速開始

麥克風在瀏覽器只能於 `https://` 或 `localhost` 使用，請在本資料夾啟動一個簡單伺服器：

```bash
python3 -m http.server 8000
```

然後開啟 <http://localhost:8000>，步驟：

1. 點「📁 選擇警報音檔（MP3）」載入你要播放的音檔。
2. 點「🎙️ 開啟麥克風並開始監聽」並允許麥克風權限。
3. 偵測到疑似哭聲時會自動播放，播放結束後自動回到監聽。

可用滑桿調整**靈敏度**與**需持續時間**（避免瞬間雜音誤觸）。

## 偵測原理

以 Web Audio API 分析麥克風輸入，符合以下條件並連續維持一段時間才觸發：

- 音量夠大聲
- 主頻率落在嬰兒哭聲範圍（約 250–700 Hz）
- 能量集中在語音頻帶（250–3000 Hz）

屬啟發式判斷，非醫療級偵測，請依實際環境調整。

## 關於 MP3 下載

原本要下載的 YouTube 音檔，因本雲端環境的網路政策封鎖了 YouTube（無法連線），
在此無法代為下載。請在自己電腦上用你慣用的工具取得該音檔後，於網頁中選擇載入即可。
